# Peinture Boisbriand

Site vitrine d'une équipe de peintres rapides à Boisbriand, Québec — spécialistes en condos, townhouses et petits commerces.

## Stack
HTML5 + CSS3 + JavaScript vanilla. Aucune dépendance.

## Sections signature
- **Délais cette semaine** — calendrier de disponibilités sur 7 jours avec slots libres / réservés / soir
- **Tarifs express** — tableau de prix annoncés à l'avance pour les chantiers fréquents
- **Process en 3 étapes** — On arrive · On peint · On part propre
- **Réservation rapide** — formulaire 3 champs

## Stack visuel
- Palette : noir #111 / orange brûlé #d96f3a / blanc
- Typo : Space Grotesk + Inter
- Transitions : snap, 100–250ms

## Déploiement (Vercel)
1. Importer ce dossier comme nouveau projet sur Vercel.
2. Aucune commande de build (site statique).
3. Pointer le domaine `peintureboisbriand.ca` vers Vercel.
